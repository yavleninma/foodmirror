# Marketing Context Pack

> **ПРАВИЛО:** Весь маркетинг должен соответствовать tone of voice продукта.
> НЕ используй: мотивацию, давление, shame, восторг, "лучшее приложение", "достигай целей".

## Стадия
Pre-launch. 0 пользователей. Нет лендинга, соцсетей, каналов.

## Позиционирование

### Одна фраза
«Зеркало питания. Без целей, без давления, без ложной точности.»

### Для кого
Взрослые, которые уже знают что такое калории и БЖУ. Которым не нужен тренер в телефоне. Которые хотят видеть факты, а не "ты молодец" или "тебе осталось 200 ккал".

### Против кого
Все фуд-трекеры (MyFitnessPal, FatSecret, Yazio, LoseIt) — они про цели, стрики, «осталось калорий». FoodMirror — единственный, который этого принципиально не делает.

### Уникальность
1. **Нет целей** — нигде, никогда
2. **Диапазоны вместо точных цифр** — честно показывает неопределённость
3. **Фото → оценка за 5 сек** — минимум действий
4. **"Почему так?"** — объяснение каждой оценки
5. **Telegram-native** — не нужно устанавливать приложение

## Tone of Voice

### ДА
- Факты без эмоций
- Краткость
- Техническая точность
- «≈ 450 ккал ±15%» вместо «примерно 450 калорий»
- Нейтральные наблюдения: «выше обычного», «ниже среднего»

### НЕТ
- "Отличный выбор!"
- "Ты справишься!"
- "Осталось X калорий"
- "Рекомендуем..."
- Восклицательные знаки
- Эмодзи (кроме 🟡 статуса)

## Ключевые сообщения для копирайта

### Hook
«Ты знаешь сколько калорий в твоей шаурме? Плюс-минус 200 ккал — и это нормально.»

### Value prop
«Сфоткал → получил оценку с диапазоном → подтвердил → понял день. Без целей и советов.»

### Дифференциатор
«Все трекеры говорят "тебе осталось 500 ккал". Мы говорим "сегодня ≈ 2100 ккал ±12%". Разница — в честности.»
