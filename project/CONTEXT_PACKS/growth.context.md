## Гипотезы привлечения (приоритизированные)

### Tier 1 — Быстрые, дешёвые
1. **Личная сеть + тематические чаты**
   - Фитнес-чаты, ЗОЖ-сообщества, чаты про питание
   - Формат: не реклама, а "сделал бота, попробуйте"
   - Метрика: первые 10-20 юзеров для валидации

2. **Product Hunt / Show HN**
   - Нишевый продукт с чёткой позицией = хороший PH кандидат
   - Подготовка: лендинг + 30-сек видео демо
   - Метрика: трафик + signups

3. **Telegram-каталоги ботов**
   - BotList, Telegram Bot Store, тематические каталоги
   - Бесплатно, low effort
   - Метрика: organic discovery

### Tier 2 — Контентные
4. **Telegram-канал бренда**
   - Контент: "шаурма = 600±120 ккал", "чем отличается ±15% от точных цифр"
   - Формат: минималистичный, факты, без мотивации
   - Метрика: подписчики → конверсия в бот

5. **Образовательный контент**
   - "Почему все калории неточные" — статья/thread
   - "Неделя с трекером без целей" — личный опыт
   - Метрика: шеры, переходы

### Tier 3 — Виральные
6. **Шеринг результатов**
   - Кнопка "Поделиться днём" → красивая картинка с итогами
   - Формат: ≈ 2150 ккал · Б 105 · Ж 90 · У 180
   - Вирусный потенциал среди ЗОЖ-аудитории
   - Метрика: шеры → новые юзеры

7. **Реферальная программа**
   - "Пригласи друга → +7 дней Premium"
   - Простая механика через Telegram deep links
   - Метрика: referral rate, K-factor

### Tier 4 — Платные (позже)
8. **Telegram Ads**
   - Таргет: фитнес-каналы, ЗОЖ
   - Бюджет: после валидации retention
