# E2E тесты FoodMirror (Telegram Web)

Запуск: браузер должен быть открыт на https://web.telegram.org/k/#@FoodMirror_test_bot, пользователь залогинен.

**UX:** В Telegram Web кнопки бота (Подтвердить, Отмена, Сегодня и т.д.) могут быть скрыты — навести курсор на значок рядом со скрепкой (слева от поля ввода), чтобы раскрыть кастомную клавиатуру.

## Подготовка

- [ ] Остановить дублирующий инстанс бота на VPS (оставить только локальный Docker)
- [ ] Убедиться: `docker ps` показывает `foodmirror-bot-1` Up

---

## Тест 1: Текстовый ввод → оценка → подтверждение

| Шаг | Действие | Ожидание |
|-----|----------|----------|
| 1 | Написать «овсянка 100г с бананом» и отправить | «Подождите» |
| 2 | Подождать 5–10 сек | «Оценка: ≈ … ккал Б: … Ж: … У: …» |
| 3 | Навести на значок рядом со скрепкой | Появляются кнопки «Подтвердить», «Отмена» |
| 4 | Нажать «Подтвердить» | «Принято.» + контекст дня (ккал, БЖУ) |
| 5 | Навести на значок | Кнопки «Сегодня», «Вчера», «Статистика» |

**Результат:** ✅ / ❌
