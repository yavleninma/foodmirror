# Модель продаж FoodMirror — выбранный вариант

> Один вариант: обоснование и разметка «бесплатно / платно».

---

## Выбранная модель: **Триал 14 дней → Freemium + Premium**

1. **Триал 14 дней** — полный доступ (в т.ч. статистика), чтобы попробовать.
2. **После триала** — бесплатно: оценка по фото/тексту без ограничений; статистика скрыта.
3. **Оплата** — снова полный доступ (статистика, напоминания).

---

## Что бесплатно / что платно

### Бесплатно (Freemium)

| Элемент | Решение |
|--------|--------|
| Оценка по фото | Без ограничений |
| Оценка по тексту | Без ограничений |
| Подтверждение приёмов | Без ограничений |
| «Почему так?» (объяснение оценки) | Бесплатно |
| Сводка после confirm | Минимальная («Принято. «Название»») — без итогов за день |

Статистика считается для всех в фоне, но **видеть её** могут только Premium. Бесплатные пользователи могут сами всё сложить, если хотят.

### Платно (Premium)

| Элемент | Решение |
|--------|--------|
| Итоги за день (Сегодня) | Да |
| Итоги за вчера (Вчера) | Да |
| Статистика 7/30 дней | Да |
| Сравнение с 7-дневным средним | Да |
| Сводка после confirm | Полная (ккал, БЖУ за день) |
| Вечернее напоминание | Да |

### Гипотеза конверсии

Триал даёт попробовать всё. За 2 недели формируется привычка. После триала статистика исчезает — это раздражает, хочется видеть итоги и тренды → платят.

---

## Реализация (соответствие кода)

- **Триал = подписка:** при первом confirm автоматически выдаётся подписка на 14 дней в `subscribedUntil`. Сообщение как при оплате/гранте: «Вам выдана подписка Premium на 14 дней».
- **Подписка:** `User.subscribedUntil`. Админ: `/admin grant <@username|chatId> <дней>` — выдача на N дней; `/admin setend`, `/admin revoke`. Оплата — 30 дней, продлевает от текущего конца (если подписка активна) или от сегодня.
- **Ограничение просмотра:** Сегодня, Вчера, Статистика — только `hasFullAccess` (= isPremium).
- **Напоминания:** за 7, 3, 1 день до конца подписки; «Подписка закончилась»; вечернее напоминание — только при активной подписке.
- **Антиспам:** не более 100 фото и 150 запросов оценок в день (все пользователи).
