# AI System — Индекс документации

Папка `ai-system/` хранит актуальный контекст проекта для разработчиков и ИИ-ассистентов.

## Быстрый вход

1. [`AGENT_QUICKSTART.md`](./AGENT_QUICKSTART.md) — минимальный обязательный контекст (5 минут)
2. [`../README.md`](../README.md) — продуктовая рамка MVP

## Когда читать что

- Архитектура и поток данных:
  [`ARCHITECTURE.md`](./ARCHITECTURE.md)
- Стек и команды:
  [`TECH_STACK.md`](./TECH_STACK.md)
- API-контракт:
  [`API_SPEC.md`](./API_SPEC.md)
- Модель данных:
  [`DATA_MODEL.md`](./DATA_MODEL.md)
- Кодстайл и правила:
  [`CONVENTIONS.md`](./CONVENTIONS.md)
- Куда движемся сейчас:
  [`ONE_PAGER.md`](./ONE_PAGER.md)
- Очередь задач:
  [`BACKLOG.md`](./BACKLOG.md)
- Принятые решения:
  [`DECISIONS.md`](./DECISIONS.md)
- Деплой:
  [`DEPLOY.md`](./DEPLOY.md)

## Правило актуальности

- Если контракт/поведение поменялось, сначала обновляем профильный документ в `ai-system/`, затем код.
- Если документ противоречит коду, источником истины считается код, но документ нужно поправить в том же PR.
