# ONE PAGER — Текущий фокус

**Обновлено:** 2026-03-02

## Что уже есть

- v2.0 монорепо (api/web/shared) на TypeScript
- PostgreSQL + Prisma, базовые модели `User`, `Insight`
- Флоу MVP: фото → AI-инсайт → история/статистика (в Telegram)
- Деплой на Railway через Dockerfile

## Что делаем сейчас

1. Guest-mode вне Telegram:
   основная функция (фото и инсайт) работает, личные функции закрыты.
2. Стабилизация API:
   исправление пограничных багов (старт недели, курсорная пагинация).
3. Базовый тестовый контур:
   первые unit-тесты по auth-mode и логике статистики.

## Главная продуктовая метрика

Среднее количество фото **до еды** на пользователя в неделю.

## North Star

Пользователь автоматически фотографирует еду до еды и чувствует контроль без подсчёта калорий.
